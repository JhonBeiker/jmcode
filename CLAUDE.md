# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project status

This repository currently contains only planning material and brand assets — no application code has been scaffolded yet. There is no `package.json`, build tooling, or test suite to run.

## Project goal

Build a landing page for **JMcode**, a company offering digital services:
- Application development
- Web page development
- Web system development

The site must be built with **Astro** and styled with **Tailwind CSS**. When scaffolding the project, use Astro's official Tailwind integration and follow Astro's standard project structure (`src/pages`, `src/components`, `src/layouts`, etc.).

## Assets

The `assets/` directory contains the JMcode brand kit:
- Logo files in multiple formats/resolutions (CMYK, RGB, PNG at various sizes) — pick the appropriate variant based on context (web vs. print, light vs. dark backgrounds).
- `Manual de Marca JMCODE.pdf` — the brand manual. Consult this for colors, typography, and logo usage rules before making design decisions on the landing page.
